# CiderScope - Overview

CiderScope (SensoPlatform) est une plateforme spécialisée dans l'analyse sensorielle pour des jurys.

## Objectif

Permettre la création de sessions de dégustation et l'analyse statistique avancée des retours des participants (tests de Friedman, Nemenyi, etc.).

## Stack Technique

- **Frontend** : Next.js 16 (App Router), React 19, TailwindCSS
- **Backend / Database** : Supabase
- **Tests** : Vitest
- **CI** : GitHub Actions

Pour en savoir plus, consultez la page [Getting Started](./getting-started.md).
